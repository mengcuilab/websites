4. Steps to install jekyll:

5. Steps to init a website from scratch:

```
  $ mkdir wireframe
  $ cd wireframe
  $ git init 
  $ bundle init
  add the line below into Gemfile
	  gem "jekyll" 
  $ bundle
  $ bundle exec jekyll serve

```
